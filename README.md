# ehgoodenough.github.io #

A website for... my portfolio, I guess?

### How to Build ###

Have node v8+ and npm v5+.

```
node --version
npm --version
```

Build to [localhost:8080](http://localhost:8080).

```
npm install
npm start
```

Deploy to [github.io](https://ehgoodenough.github.io).

```
npm run deploy
```
